# NameOps

Name clearance orchestrator — batch runs, publish, and PR automation.

## Key Features

- **Batch Clearance** — Check multiple product/project names against clearance criteria
- **CI Integration** — Runs as a GitHub Actions workflow with artifact upload
- **PR Automation** — Generates PR body with clearance results
- **Caching** — Persistent cache to avoid redundant checks
